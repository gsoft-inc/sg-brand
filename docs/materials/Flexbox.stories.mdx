# Flexbox

<Tagline>Flexbox helps achieve powerful horizontal or vertical layouts without JavaScript.</Tagline>

Before using these utilities, you should be familiar with CSS3 Flexible Box specifications. If you are not, check out this [CSS-Tricks guide](https://css-tricks.com/snippets/css/a-guide-to-flexbox/). You will find most of the documentation for flexbox utilities on [Tachyons website](https://tachyons.io/docs/layout/flexbox/).

For an exhaustive list of utilities classes for flexbox **including our customizations**, please refer to the [Tachyons section](#tachyons) below.

## Utilities

A few flexbox utilities have been added to Tachyons for common use cases that we kept writing specific selectors for.
