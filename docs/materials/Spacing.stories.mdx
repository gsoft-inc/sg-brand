# Spacing

Orbit Design spacing scale use an 8 point scale.

We chose a base-8 scale because eight is a highly composable number (it can be divided and multiplied many times and result in whole numbers), yet allows spacing dense enough for ShareGate's products.

A few half points have been added to cover specific use cases that requires smaller padding and margin.

### Units

Our spacing values are expressed in `rem` units based on a font-size of `16px`. We choosed `rem` units over `em` units since they are relative to the font-size of the root element instead of the parent element which can cause unexpected results when cascading.

For a deep understanding of rem units, refer to the [Values and units](https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Values_and_units) article on MDN.

## Utilities

Orbit UI spacing utilities classes includes [margin](https://tachyons.io/docs/layout/spacing/), [padding](https://tachyons.io/docs/layout/spacing/), [width](https://tachyons.io/docs/layout/widths/), [max-width](https://tachyons.io/docs/layout/max-widths/) and [height](https://tachyons.io/docs/layout/heights/).

You will find most of the documentation for spacing utilities on [Tachyons website](https://tachyons.io/docs/layout/spacing/). Keep in mind though that the spacing scale have been increase to 13 values to match Orbit Design.

For an exhaustive list of utilities classes for spacing **including our customizations**, please refer to the [Tachyons section](#tachyons) below.

## Paddings

Selectors for `auto` paddings have been added to Tachyons: `pa-auto`, `pl-auto`, `pr-auto`, `pt-auto`, `pb-auto`, `pv-auto`, `ph-auto`.

## Margins

Selectors for `auto` margins have been added to Tachyons: `ma-auto`, `ml-auto`, `mr-auto`, `mt-auto`, `mb-auto`, `mv-auto`, `mh-auto`.
