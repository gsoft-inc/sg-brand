# Colors

ShareGate colors are meant to put forward our brand and to express our personality in a constantly growing market.

Any of the following colors value can be used in two ways, either as a [style prop](?path=/docs/style-props--page):

```jsx
<Div backgroundColor="alias-accent" color="purple-1">To the moon!</Div>
```

or as a CSS variable:

```css
.container {
    background-color: var(--o-ui-bg-alias-accent);
    color: var(--o-ui-purple-1);
}
```

## Semantic colors

Semantic colors, also known as *aliases* are colors with meaningful names representing the general realm of their usage. They provide an additional layer of abstraction on top of the [color palettes](#color-palettes) which allow Orbit to seamlessly support multiple [color schemes](?path=/docs/color-schemes--page).

This means that every semantic color map to a value of an Orbit [color palette](#color-palettes). The value might vary depending of the active color scheme.
