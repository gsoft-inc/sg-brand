# Semantic Colors

<Tagline>Semantic colors are a way to use colors using meaningful names. This serves two purposes, it helps deginers and developers decide what colors to use and makes our color system more efficient and flexible.</Tagline>

### Usage

Semantic colors also known as aliases can be used in two ways, either in a CSS in JS prop value, e.g. `backgroundColor="alias-accent"` or in CSS, as a variable `background-color: var(--o-ui-bg-alias-accent)`.

When used in a Styled System prop there's no need to specify what property it affects as Styled System is smart enough to do it for you. This means that you only need to specify that the value you are using is an alias and which alias you want to use.
