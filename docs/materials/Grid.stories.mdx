# Grid

<Tagline>Grid helps achieve powerful horizontal or vertical layouts without JavaScript.</Tagline>

Before using these utilities, you should be familiar with CSS3 Grid specifications. If you are not, check out this [CSS-Tricks guide](https://css-tricks.com/snippets/css/complete-guide-grid/). Grid utilities have been added to Tachyons for common use cases that we kept writing specific selectors for

For an exhaustive list of utilities classes for grid, please refer to the [Tachyons section](#tachyons) below.

## Utilities

<ExternalLink href="https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Grid_Layout">CSS Grid Layout</ExternalLink> excels at dividing a page into major regions or defining the relationship in terms of size, position, and layer, between parts of a control built from HTML primitives. We provide utilities that helps aligning grid items, we do not intend to provide an utility for all occassions, grid being complex.
