# Foundation

Orbit UI foundation contains the [CSS variables](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties) that store the visual design attributes of Orbit Design. We use those variables across Orbit UI instead of hard-coded values in order to maintain a scalable and consistent visual system for UI development.

## Use variables

A foundation CSS variable can be accessed in a CSS class:

```css
.button {
    color: var(--o-ui-global-sunray-50);
}
```

In a `style` attribute:

```html
<button style="color: var(--o-ui-global-sunray-50)" type="button">
    Click me!
</button>
```

In JavaScript (the computed variable value is available as a string):

```javascript
window.getComputedStyle(document.documentElement).getPropertyValue("--o-ui-global-sunray-50");
```

## Override variables

Once installed the default values of the CSS foundation can be customized to your needs by simply redeclaring the variables at the `.o-ui` class or lower in the cascade needed :

```css
.o-ui {
    --o-ui-alias-primary-50: #c63a0c;
    --o-ui-global-apricot-50: #c63a0c;
    --o-ui-global-desktop-50: #c63a0c;
}
```

If you need to use a dynamic value or to scope an update to a specific element you can override a variable with a `style` attribute:

```html
<button style="--o-ui-sunray-50: #c63a0c" type="button">
    Click me!
</button>
```

At last resort, you can override a variable in JavaScript. However, we strongly encourage using a CSS class or a `style` attribute.

```javascript
document.body.style.setProperty("--o-ui-sunray-50", "#c63a0c");
```

> Please be aware that customization would no longuer benefits from updates and could break.
