# Packages

Multiple packages compose Orbit UI. Based on which type of application you're building you'll choose a different composition. For more information view the [installation page](?path=/docs/installation--page).

## Bundles

Bundles are the easiest way to install Orbit UI in your app with a single package.

    packages={[

The core packages are the backbone of Orbit UI. They include all the Orbit UI bits that are related to styling.
