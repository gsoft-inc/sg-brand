# Theme Reference

## Keys

### Space

The [space](/story/spacing--page) key is a specially-named scale (scale 13) intended to be used with `margin`, `padding`, and other layout-related CSS properties.

### Sizing

The [sizing](/story/sizing--page) key is a specially-named scale (scale 18) intended to be used with `width`, `height`, and other layout-related CSS properties.

### Colors

[Colors](/story/colors--page) provided by the theme are intended to be used with `color`, `background-color` and other color related CSS properties. They provides light and dark variation of semantic colors.

The following is a list of theme object keys and their corresponding CSS properties. This list may be non-exhaustive.
