# Label

## Installation

<details>
    <summary >install the packages</summary>
    Install from <a href={INSTALLATION_PAGE}>the React bundle</a> or as a single package:
    ```bash
    npm install @orbit-ui/css-normalize @orbit-ui/tachyons @orbit-ui/semantic-ui-theme semantic-ui-react
    ```
</details>

<details>
    <summary >then import what you need</summary>
    ```javascript
        import { Button, Icon, Label } from "semantic-ui-react";
    ```
</details>

## Usage

### Default

A basic label.

<Preview>
    <Story name="default">
        <Label>Notification Sent</Label>
    </Story>
</Preview>

### Icons

A label can include icons.

<Preview>
    <Story name="icons">
        <div className="flex">
            <Label>
            Notification Sent <Icon name="search" />
            </Label>
            <Label>
                <CommunicationIcon className="h6 w6 fill-marine-500 mr1" />
        Notification Sent <Icon name="search" />
            </Label>
        </div>
    </Story>
</Preview>

### Sizes

A label can vary in size.

<Preview>
    <Story name="sizes">
        <div className="flex items-start">
            <Label size="small">Notification Sent</Label>
            <Label size="large">Notification Sent</Label>
        </div>
    </Story>
</Preview>

### Circular

A label can be circular.

<Preview>

A label can reduce its complexity.

## Variations

### Tags

A tag consist of an icon/label in a label.

<Preview>
    <Story name="tag default">
        <Label>
            <Label circular empty size="mini" color="red" />
            Group added
        </Label>
    </Story>
</Preview>

#### Icons

    <Story name="tag icons">
