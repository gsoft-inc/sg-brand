# Dropdown

A dropdown can be used to select between choices in a form.

## Installation

<details>
    <summary >install the packages</summary>
    Install from <a href={INSTALLATION_PAGE}>the React bundle</a> or as a single package:
    ```bash
    npm install @orbit-ui/css-normalize @orbit-ui/tachyons @orbit-ui/semantic-ui-theme semantic-ui-react
    ```
</details>

<details>
    <summary >then import what you need</summary>
        ```javascript
            import { Dropdown } from &quot;semantic-ui-react&quot;;
        ```
</details>
