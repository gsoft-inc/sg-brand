## Font Weights

Varying the font-weight of different pieces of text can help create contrast between pieces of information.

| Value | Class |
|-:-----|-:-----|
| 100   | fw1   |
| 300   | fw3   |
| 400   | fw4   |
| 500   | fw5   |
| 600   | fw6   |
| 700   | fw7   |
| 900   | fw9   |
