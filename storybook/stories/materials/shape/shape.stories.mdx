## Radiuses

An element can have a border radius modifying its shape.

| Scale | Value |
| :---- | :---- |
| 1     | 2px   |
| 2     | 4px   |
| 3     | 8px  |
| 4     | 16px  |
| pill  | 999px |
| 100   | 100%  |
| 0     | 0     |
