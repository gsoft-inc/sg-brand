# Shadows

Shadows denote a certain elevation on an element.

## Installation

<details><summary className="outline-0">Install the packages</summary>

Install from <a href={INSTALLATION_PAGE}>the React bundle</a> or as a single package:

```bash
npm install @orbit-ui/css-normalize @orbit-ui/tachyons
```
