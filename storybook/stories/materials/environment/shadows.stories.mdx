# Shadows

Shadows denote a certain elevation of an element.

## Installation

<details><summary >Install the packages</summary>

Install from <a href={INSTALLATION_PAGE}>the React bundle</a> or as a single package:

```bash
npm install @orbit-ui/css-normalize @orbit-ui/tachyons
```

</details>

| 200     | rgba(128, 136, 157, 0.32) 0px 8px 24px 0px | shadow-200 |
