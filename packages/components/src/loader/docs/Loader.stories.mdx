## Guidelines

### Accessibility
- The loader has a [role](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles) of `status` by default.

### Loader vs Spinner
- When you need to give feedback for a user or system action, use a [spinner](?path=/docs/spinner--default-story) instead.

## Usage

A default loader. Use in most situations.

### Delay

By default, a loader can appear after a short delay. This is to prevent the loader from appearing when the action is very fast.
If you want to disable this delay, you can set the `delay` prop to `null`. You can also set a custom delay in milliseconds.

<Preview>
    <Story name="delay">
        <Inline alignY="end">
            <Loader delay={null} aria-label="Loading..."/>
        </Inline>
    </Story>
</Preview>
