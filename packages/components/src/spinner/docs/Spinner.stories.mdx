# Spinner

## Guidelines

### Accessibility
- The spinner has a [role](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles) of `status` by default. When inside a parent component that is focusable, the spinner should have a role of `presentation` to prevent the screen reader from reading the spinner.

### Spinner vs Loader
- When you need to give feedback for a page or subset of the page that is loading, use a [loader](?path=/docs/loader--default-story) instead.
