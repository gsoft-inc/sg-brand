# Lozenge

<Tagline>Lozenges are used to highlight an item's status for quick recognition or to give additional fast short information such as highlighting a new item.</Tagline>

<ComponentInfo
    usage={"import { Lozenge } from \"@sharegate/orbit-ui\";"}
    githubPath="/packages/components/src/lozenge/src"
/>

## Usage

### Default

A default lozenge.

<Preview>
    <Story name="default">
        <Lozenge>New</Lozenge>
    </Story>
</Preview>

### Icon

A lozenge can contain an [icon](?path=/docs/icon-gallery--page).

<Preview>
    <Story name="icon">
        <Lozenge>
            <CheckCircleIcon />
            <Text>New</Text>
        </Lozenge>
    </Story>
</Preview>

### Size

A lozenge can vary in size.

<Preview>
    <Story name="size">
        <Inline alignY="center">
            <Lozenge size="sm">New</Lozenge>
            <Lozenge>New</Lozenge>
        </Inline>
    </Story>
</Preview>


### Highlight

A lozenge can be highlighted.

<Preview>
    <Story name="highlight">
        <Inline alignY="center">
            <Lozenge size="sm" variant="highlight">New</Lozenge>
            <Lozenge variant="highlight">New</Lozenge>
        </Inline>
    </Story>
</Preview>

### Warning

A lozenge can use a warning color.

<Preview>
    <Story name="warning">
        <Inline alignY="center">
            <Lozenge tone="warning">Expires soon</Lozenge>
            <Lozenge tone="warning" variant="highlight">Expires soon</Lozenge>
        </Inline>
    </Story>
</Preview>

### Positive

A lozenge can use a positive color.

<Preview>
    <Story name="positive">
        <Inline alignY="center">
            <Lozenge tone="positive">Active</Lozenge>
            <Lozenge tone="positive" variant="highlight">Active</Lozenge>
        </Inline>
    </Story>
</Preview>


### Negative

A lozenge can use a positive color.

<Preview>
    <Story name="negative">
        <Inline alignY="center">
            <Lozenge tone="negative">Blocked</Lozenge>
            <Lozenge tone="negative" variant="highlight">Blocked</Lozenge>
        </Inline>
    </Story>
</Preview>
