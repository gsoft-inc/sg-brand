# Alert

<Tagline>An alert is an element that displays a brief, important message in a way that attracts the user's attention without interrupting the user's task.</Tagline>

<ComponentInfo
    usage={"import { Alert, Content } from \"@orbit-ui/react-components\";"}
    ariaPath="#alert"
    githubPath="/packages/react-components/src/alert/src"
/>

## Usage

### Default

A default alert.

<Preview>
    <Story name="default">
        <Alert>Your email to <strong>booking@spacetravel.com</strong> has been sent!</Alert>
    </Story>
</Preview>

### Icon

An alert can have an [icon](?path=/docs/icon-gallery--page).

<Preview>
    <Story name="icon">
        <Alert>
            <MailIcon />
            <Content>Your email to <strong>booking@spacetravel.com</strong> has been sent!</Content>
        </Alert>
    </Story>
</Preview>

### Rich content

An alert can also accept rich content.

<Preview>
    <Story name="rich content">
        <Alert>
            <MailIcon />
            <Content>
                <Heading>booking@spacetravel.com</Heading>
                Your email has been sent! Tracking info available <TextLink href="#">here</TextLink>.
            </Content>
        </Alert>
    </Story>
</Preview>

### Action

An alert can accept an accept [button](?path=/docs/button--default-story).

<Preview>
    <Story name="action">
        <Alert>
            <MailIcon />
            <Content>Your email to <strong>booking@spacetravel.com</strong> has been sent!</Content>
            <Button>Undo</Button>
        </Alert>
    </Story>
</Preview>

### Dismissible

An alert can be dismissible by specifying an `onDismiss` handler.

<Preview>
    <Story name="dismissible">
        <Alert onDismiss={() => {}}>
            <MailIcon />
            <Content>Your email to <strong>booking@spacetravel.com</strong> has been sent!</Content>
            <Button>Undo</Button>
        </Alert>
    </Story>
</Preview>

### Info

Use an info alert to notify a user of a change that's relevant for them to achieve their current goal.

<Preview>
    <Story name="info">
        <InfoAlert onDismiss={() => {}}>
            <Content>Your email to <strong>booking@spacetravel.com</strong> has been sent!</Content>
            <Button>Undo</Button>
        </InfoAlert>
    </Story>
</Preview>

### Positive

Use a positive alert to inform a user that an action has completed successfuly.

<Preview>
    <Story name="positive">
        <PositiveAlert onDismiss={() => {}}>
            <Content>Your email to <strong>booking@spacetravel.com</strong> has been sent!</Content>
            <Button>Undo</Button>
        </PositiveAlert>
    </Story>
</Preview>

### Warning

Use a warning alert to help a user avoid an issue.

<Preview>
    <Story name="warning">
        <WarningAlert onDismiss={() => {}}>
            <Content>We are having trouble contacting the email server.</Content>
            <Button>Retry</Button>
        </WarningAlert>
    </Story>
</Preview>

### Critical

Use an error alert to help a user understand a critical issue at the system level.

<Preview>
    <Story name="critical">
        <CriticalAlert onDismiss={() => {}}>
            <Content>Cannot contact email server.</Content>
            <Button>Retry</Button>
        </CriticalAlert>
    </Story>
</Preview>

### Controlled

An alert can handle `show` state in controlled mode.
