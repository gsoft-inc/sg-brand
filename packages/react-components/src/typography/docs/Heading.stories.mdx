# Heading

<Tagline>Heading is used to create various levels of typographic hierarchies.</Tagline>

## Usage

### Element type

When using an header, you should specify the element type to render it as. We do recommended using native header elements `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>`.

<Preview>
    <Story name="element type">
        <Heading as="h1">Houston, Tranquillity Base here. The Eagle has landed.</Heading>
    </Story>
</Preview>

### Size

You can alter the size of the heading by specifying a `size` prop. The available size match Orbit UI [typographic headlines](?path=/docs/typography--page#headlines).

<Preview>
    <Story name="size">
        <>
            <Heading as="h3">Houston, Tranquillity Base here. The Eagle has landed.</Heading>
            <Heading as="h4" size="sm">Houston, Tranquillity Base here. The Eagle has landed.</Heading>
        </>
    </Story>
</Preview>

### Aliases

Convenience aliases are available for standard HTML headers.

<Preview>
    <Story name="aliases">
        <>
            <H1>Houston, Tranquillity Base here. The Eagle has landed.</H1>
            <H2>Houston, Tranquillity Base here. The Eagle has landed.</H2>
            <H3>Houston, Tranquillity Base here. The Eagle has landed.</H3>
            <H4>Houston, Tranquillity Base here. The Eagle has landed.</H4>
            <H5>Houston, Tranquillity Base here. The Eagle has landed.</H5>
            <H6>Houston, Tranquillity Base here. The Eagle has landed.</H6>
        </>
    </Story>
</Preview>
