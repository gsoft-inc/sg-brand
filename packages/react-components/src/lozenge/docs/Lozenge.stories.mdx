# Lozenge

<Tagline>Lozenges are used to highlight an item's status for quick recognition or to give additional fast short information such as highlighting a new item.</Tagline>

<ComponentInfo
    usage={"import { Lozenge } from \"@orbit-ui/react-components\";"}

## Usage

### Default

A default lozenge.

<Preview>
    <Story name="default">
        <Lozenge>New</Lozenge>
    </Story>
</Preview>

### Icon

A lozenge can contain an [icon](?path=/docs/icon-gallery--page).

<Preview>
    <Story name="icon">
        <Lozenge>
            <CheckCircleIcon />
            <Text>New</Text>
        </Lozenge>
    </Story>
</Preview>

### Size

A lozenge can vary in size.

<Preview>
    <Story name="size">
        <Inline verticalAlign="center">
            <Lozenge size="sm">New</Lozenge>
            <Lozenge>New</Lozenge>
        </Inline>
    </Story>
</Preview>

### Color

A lozenge can use a different color accent.

<Preview>
    <Story name="color">
        <Lozenge color="primary">New</Lozenge>
    </Story>
</Preview>
