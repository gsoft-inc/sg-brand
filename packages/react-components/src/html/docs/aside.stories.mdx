## Usage

### Example

An aside component accepts all the [HTML element attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/aside) and Orbit styled system props.

```jsx
<Aside color="cloud-8">
    <Paragraph>The Universe is under no obligation to make sense to you. The Earth is the cradle of humanity, but mankind cannot stay in the cradle forever.</Paragraph>
</Aside>
```
