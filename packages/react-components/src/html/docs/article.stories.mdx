## Usage

### Example

An article component accepts all the [article HTML element attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/article) and Orbit styled system props.

```jsx
<Article color="cloud-8">
    <H1>Weather forecast for Boca Chica</H1>
    <Article>
        <H2>03 March 2021</H2>
        <Paragraph>Perfectly sunny for a space launch.</Paragraph>
    </Article>
</Article>
```
