## Usage

### Example

A nav component accepts all the [nav HTML element attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/nav) and Orbit styled system props.

```jsx
<Nav color="cloud-8">
    <UL>
        <LI><TextLink href="#">Missions</TextLink></LI>
        <LI><TextLink href="#">Launches</TextLink></LI>
        <LI><TextLink href="#">Careers</TextLink></LI>
    </UL>
</Nav>
```
