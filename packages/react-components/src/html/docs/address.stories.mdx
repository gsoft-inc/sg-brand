## Usage

### Example

An address component accepts all the [HTML element attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/address) and Orbit styled system props.

<Preview>
    <Story name="example">
        <Address color="cloud-8">
            <A href="mailto:media@spacex.com">media@spacex.com</A>
            <br />
            <A href="tel:+13103636000">(310) 363-6000</A>
        </Address>
    </Story>
