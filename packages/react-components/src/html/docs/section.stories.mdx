## Usage

### Example

A section component accepts all the [HTML element attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/section) and Orbit styled system props.

```jsx
<Body>
    <H1>Blue Origin</H1>
    <Section color="cloud-10">
        <H2>Our vision</H2>
        <P>Blue Origin was founded by Jeff Bezos with the vision of enabling a future where millions of people are living and working in space to benefit Earth.</P>
    </Section>
</Body>
```
