## Usage

### Example

A body component accepts all the [HTML element attributes](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/body) and Orbit styled system props.

```jsx
<html>
    <head>
        <title>Space diary</title>
    </head>
    <Body color="cloud-10">
        <Paragraph>The Universe is under no obligation to make sense to you. The Earth is the cradle of humanity, but mankind cannot stay in the cradle forever.</Paragraph>
    </Body>
</html>
```
