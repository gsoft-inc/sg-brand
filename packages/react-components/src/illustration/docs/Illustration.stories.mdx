# Illustration

<Tagline>Use an illustration to compose an image with a background color, some text or wathever else you need.</Tagline>

## Usage

### Horizontal

An illustration can render horizontally.

<Preview scope={{ NoResults }}>
    <Story name="horizontal">
        <Illustration>
            <SvgImage src={NoResults} stroke="alias-icon-1" />
            <Heading>Can't find "Saturn"</Heading>
            <Content>Please try another search</Content>
        </Illustration>
    </Story>
</Preview>

### Vertical

An illustration can render vertically.

<Preview scope={{ Apollo11Badge }} style={{ height: "500px" }}>
    <Story name="vertical">
        <Illustration orientation="vertical" color="#E7BE68" style={{ width: "225px", padding: "24px" }}>
            <Image src={Apollo11Badge} alt="Apollo 11 poster" />
        </Illustration>
    </Story>
</Preview>

### Shape

An illustration can have a `rounded` shape.

<Preview scope={{ Apollo11Badge }}>
    <Story name="shape">
        <Illustration shape="rounded" color="#E7BE68" style={{ height: "225px", padding: "24px" }}>
            <Image src={Apollo11Badge} alt="Apollo 11 poster" />
        </Illustration>
    </Story>
</Preview>

### Color

An illustration backgroud color can be a name from our foundations (e.g. `sunray-50`, `primary-500`), an hexadecimal color code (e.g. `#c5cac8`), an `hsl`, an `hsla`, a `rgb` or a `rgba` color code.

<Preview scope={{ NoResults }}>
    <Story name="color">
        <Illustration color="alias-background-info-1" style={{ height: "300px" }}>
            <SvgImage src={NoResults} stroke="alias-icon-1" />
            <Heading>Can't find "Saturn"</Heading>
            <Content>Please try another search</Content>
        </Illustration>
    </Story>
</Preview>
