# Illustration

<Tagline>Use this component to compose an illustration with optionally a background color and some content.</Tagline>

    usage={"import { Illustration } from \"@orbit-ui/react-components\";"}

## Usage

### Horizontal

An illustration can render horizontally.

<Preview scope={{ NoResults }}>
    <Story name="horizontal">
        <Illustration>
            <Image src={NoResults} alt="No Results" />
            <Heading>Can't find "Saturn".</Heading>
            <Content>Please try another search.</Content>
        </Illustration>
    </Story>
</Preview>

### Vertical

An illustration can render vertically.

<Preview scope={{ Storage }} style={{ height: "400px" }}>
    <Story name="vertical">
        <Illustration orientation="vertical" color="primary-50" style={{ width: "256px" }}>
            <Image src={Storage} alt="Storage" width="160px" />
        </Illustration>
    </Story>
</Preview>

### Shape

An illustration can use a different shape.

<Preview scope={{ Storage }} style={{ height: "400px" }}>
    <Story name="shape">
        <Inline style={{ height: "100%" }}>
            <Illustration shape="straight" orientation="vertical" color="primary-50" style={{ width: "256px" }}>
                <Image src={Storage} alt="Storage" width="160px" />
            </Illustration>
            <Illustration shape="rounded" orientation="vertical" color="primary-50" style={{ width: "256px" }}>
                <Image src={Storage} alt="Storage" width="160px" />
            </Illustration>
        </Inline>
    </Story>
</Preview>

### Color

An illustration backgroud color can be a name from our foundations (e.g. `sunray-50`, `primary-500`), an hexadecimal color code `#c5cac8`, an `hsl`, an `hsla`, a `rgb` or a `rgba` color code.

<Preview scope={{ Storage }} style={{ height: "400px" }}>
    <Story name="color">
        <Illustration color="hsla(174, 61%, 94%, 1)" orientation="vertical" style={{ width: "256px" }}>
            <Image src={Storage} alt="Storage" width="160px" />
        </Illustration>
    </Story>
</Preview>
