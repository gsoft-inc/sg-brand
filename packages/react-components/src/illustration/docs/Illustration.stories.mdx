# Illustration

<Tagline>Use this component to compose an illustration with optionally a background color and some content.</Tagline>

<ComponentInfo
    usage={"import { Illustration } from \"@orbit-ui/react-components\";"}
    githubPath="/packages/react-components/src/illustration/src"

## Usage

### Horizontal

An illustration can render horizontally.

<Preview scope={{ NoResults }}>
    <Story name="horizontal">
        <Illustration>
            <Image src={NoResults} alt="No Results" />
            <Heading>Can't find "Saturn"</Heading>
            <Content>Please try another search</Content>
        </Illustration>
    </Story>
</Preview>

### Vertical

An illustration can render vertically.

<Preview scope={{ Apollo11Badge }} style={{ height: "500px" }}>
    <Story name="vertical">
        <Illustration orientation="vertical" color="#A7D2CB" style={{ width: "225px" }}>
            <Image src={Apollo11Badge} alt="Apollo 11 poster" width="175px" />
        </Illustration>
    </Story>
</Preview>

### Shape

An illustration can have a `rounded` shape.

<Preview scope={{ Apollo11Badge }}>
    <Story name="shape">
        <Illustration shape="rounded" color="#A7D2CB" style={{ height: "225px" }}>
            <Image src={Apollo11Badge} alt="Apollo 11 poster" width="175px" />
        </Illustration>
    </Story>
</Preview>

### Color

An illustration backgroud color can be a name from our foundations (e.g. `sunray-50`, `primary-500`), an hexadecimal color code (e.g. `#c5cac8`), an `hsl`, an `hsla`, a `rgb` or a `rgba` color code.

<Preview scope={{ NoResults }}>
    <Story name="color">
        <Illustration color="hsla(222, 100%, 96%, 1)" style={{ height: "300px" }}>
            <Image src={NoResults} alt="No Results" />
            <Heading>Can't find "Saturn"</Heading>
            <Content>Please try another search</Content>
        </Illustration>
    </Story>
</Preview>
