## Accessibility

An image must have an `alt` attribute.

## Usage

### Shape

An image can use a different shape.

<Preview scope={{ LaunchImage }}>
    <Story name="shape">
        <Inline>
            <Image shape="straight" src={LaunchImage} alt="SpaceX launch" width="100px" height="100px" />
            <Image shape="rounded" src={LaunchImage} alt="SpaceX launch" width="100px" height="100px" />
            <Image shape="circular" src={LaunchImage} alt="SpaceX launch" width="100px" height="100px" />
        </Inline>
    </Story>
</Preview>

### Size

An image can adjust it's size by using the `size` prop or the native `width` and `height` props.

<Preview scope={{ LaunchImageLg }}>
    <Story name="size">
        <Image size="200px" src={LaunchImageLg} alt="SpaceX launch" />
    </Story>
</Preview>

### Retina

You can let the browser decide which image is best to serve according to the user device screen pixel density.

It is highly recommended to serve a _1x_ image as well as a _2x_ image, twice the intended size. This assures the user has the best looking image possible.

_Avoid serving images that are unecessary big, images should be resized to the intended final image display size. This assures we don't waste bandwith for the user._

<Preview scope={{ LaunchImage, LaunchImage2x }}>
    <Story name="retina">
        <Image size="100px" srcSet={`${LaunchImage} 1x, ${LaunchImage2x} 2x`} alt="SpaceX launch" />
    </Story>
</Preview>
