## Usage

### Example

A section component *doesn't render anything*, but provide information about the section to it's parent container.

```jsx
<Section title="Favorite Planets">
    <Item key="earth">Earth</Item>
    <Item key="mars">Mars</Item>
    <Item key="saturn">Saturn</Item>
</Section>
```
