# Counter

<Tagline>A counter indicate a running tally.</Tagline>

<ComponentInfo
    usage={"import { Counter } from \"@orbit-ui/react-components\";"}
    githubPath="/packages/react-components/src/counter"
/>

> A counter is usually embedded in another component such as a [tag](?path=/docs/tag--default-story) or a [checkbox](?path=/docs/checkbox--default-story).

## Usage

### Default

A default counter.

<Preview>
    <Story name="default">
        <Counter>15</Counter>
    </Story>
</Preview>

### Divider

A counter can be rendered as a divider.

<Preview>
    <Story name="divider">
        <Inline verticalAlign="center">
            <Counter variant="divider">100+</Counter>
        </Inline>
    </Story>
</Preview>

### Highlight

A counter value can be highlighted.

<Preview>
    <Story name="highlight">
        <Inline gap={10} verticalAlign="center">
            <Counter highlight>15</Counter>
            <Counter highlight variant="divider">100+</Counter>
        </Inline>
    </Story>
</Preview>

### Pushed

A counter can pushed itself away from content.

<Preview>
    <Story name="pushed">
        <Inline gap={10} verticalAlign="center">
            <Box>
                <Text>Satellites</Text> <Counter pushed>15</Counter>
            </Box>
            <Box>
                <Text>Satellites</Text> <Counter pushed variant="divider">100+</Counter>
            </Box>
        </Inline>
    </Story>
</Preview>
